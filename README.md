# 1-PROGRAMACAO_AUTOMACAO

Este repositório centraliza o ecossistema completo de estudos e automações, cobrindo desde linguagens base e back-end até engenharia de dados, front-end e web analytics.

## 📁 Estrutura de Diretórios

* **01_Linguagens_Base_e_BackEnd** (Python, JavaScript_BackEnd, Assembly, C, C++, C#, Java, Php)
* **02_Google_Workspace_e_Automacao** (AppScript_Console, automacao_Planilhas, automacao_Docs, etc.)
* **03_Banco_de_Dados_e_BI** (MySQL, PowerBI)
* **04_Web_e_FrontEnd** (Html, Css, JavaScript_FrontEnd)
* **05_WEb_Analytics** (Web_Analytics)
